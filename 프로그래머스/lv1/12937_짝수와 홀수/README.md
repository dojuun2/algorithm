# [Lv. 1] 짝수와 홀수
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12937?language=python3) 

### 문제 설명
정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.
<br/><br/>

### 제한 사항
- `n`은 int 범위의 정수입니다.
- 0은 짝수입니다.
<br/><br/>

### 입출력 예
| num | return |
|:------:|:------:|
|   3    | "Odd" |
|   4   | "Even" |
